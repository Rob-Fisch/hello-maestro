# OpusMode Pricing Tiers — Feature Limits Reference

> Quick reference for feature limits by tier. For strategy and philosophy, see [PRODUCT_STRATEGY.md](file:///Users/robfisch/Documents/OpusMode/docs/strategy/PRODUCT_STRATEGY.md).

**Pricing (Launch)**:
- **Free**: $0
- **Pro**: $9.99/month or $99/year
- **Pro+**: $19.99/month or $199/year
